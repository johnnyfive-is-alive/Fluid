🚀 Getting Started
  1. Create Item Types: Define categories like STATION, RESOURCE, UNIT
  2. Add Items: Create items and assign them to types
  3. Add Characteristics: Describe your items with key-value properties
  4. Set Loading Percentages: Use the grid to manage monthly loading data
📖 Database Structure
Tables:
  • itemtypes - Categories for items
  • items - The items being tracked
  • itemcharacteristics - Key-value properties
  • itemloading - Monthly loading percentages
Relationships:
  • Items belong to one type
  • Characteristics belong to one item
  • Loading data belongs to one item
  • Deleting items cascades to related data
