Getting Started
  1. Create Item Types: Define categories like STATION, RESOURCE, UNIT, PRODUCT
  2. Add Items: Create items and assign them to types (stations, resources, etc.)
  3. Add Products: Define products that represent what your items work on
  4. Map Products to Items: Specify which items can work on which products
  5. Add Characteristics: Describe your items with key-value properties (optional)
  6. Set Loading Allocations: Use the grid to manage monthly loading by product
Database Structure
Tables:
  • itemtypes - Categories for items
  • items - The items being tracked
  • itemcharacteristics - Key-value properties
  • itemloading - Monthly loading by product
  • item_product_map - Item-product relationships
