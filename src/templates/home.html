Getting Started
  1. Add Items: Create items and assign them to types (stations, resources, etc.)
  2. Add Products: Define products that represent what your items work on
  3. Map Products to Items: Specify which items can work on which products
  4. Add Characteristics: Describe your items with key-value properties (optional)
  5. Set Loading Allocations: Use the grid to manage monthly loading by product
